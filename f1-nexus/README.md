# F1 Nexus — Full Stack Analytics App

A full-stack Formula 1 analytics platform built with Next.js, FastAPI, and real-time data integrations.

## Tech Stack

- **Frontend**: Next.js 16 (App Router), React 19, TypeScript, Tailwind CSS v4
- **Backend**: FastAPI (Python), Jolpica/Ergast API, OpenF1, Ollama (AI Analyst)
- **Deployment**: Vercel (frontend), local/cloud FastAPI backend

## Features

- **Hub** — Live news ticker, next race countdown, championship standings
- **Telemetry** — Lap-by-lap race position chart, driver telemetry
- **Strategy** — Race strategy & pit stop analysis
- **Analyst** — AI-powered race analysis powered by Ollama
- **Predictor** — Race outcome predictions with model accuracy tracking
- **History** — All-time records, legends, constructor history
- **Paddock** — 2026 driver & team profiles with career timelines

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
